The text after the `$GAP-CLOSE` skill is a gap-analyisis.  This gap analyis describes the work that needs to be done to close the functionality gap  

# Requred Steps
1.  Implement the code change described in the gap-analysis
2.  Write automated tests that assert the functionality's behavior or, if there are existing tests that cover this functionality, update the existing tests to reflect the changes
3. VERY IMPORTANT:  **only** fix what is described in the "Desired Functionality" and the gap analysis plan.  Do not attempt to fix or add functionality for anything that is not described.


# Required Coding Principles
When implementing both the gap closing code changes and automated tests, follow all principles in [instructions--coding_principles.md](../../../.ai_coding/ai_coding_common/instructions--coding_principles.md)
