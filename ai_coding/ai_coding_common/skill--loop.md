The todo file will contain 
1.  one or more tasks (or skills) to execute 
2.  `done` criteria

You job is to execute the tasks defined in step #1 and, when complete, evaluate the `done` criteria from step #2.  You may only finish when the `done` criteria from step #2 is **FULLY** achieved

recommended todo format
```
$LOOP
1. $IMPL the eval recommendations if any exist (skip on first run)
2. $EVAL
<stuff to eval>

# DONE Criteria
You are done when the $EVAL lines returns there's no work work to do
```
